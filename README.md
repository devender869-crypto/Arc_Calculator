# Arc_Calculator

A simple calculator web application built using HTML, CSS, and JavaScript.

## Features

- Supports addition, subtraction, multiplication, division, and percentage
- Shows results while entering values
- Clear and delete buttons
- Keyboard support for numbers and operators
- Responsive design for mobile and desktop screens

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open `calculator.html` with Live Server, or open it directly in a web browser.

## Keyboard Controls

- `Enter` or `=` — calculate
- `Backspace` — delete the last value
- `Esc` — clear the calculator
